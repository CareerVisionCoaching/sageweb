# Image Assets

This directory contains all image assets for the Sage.coach website.

## Current Assets (Placeholders)

All images in this directory are **SVG-based placeholders** for development and testing purposes. They should be replaced with actual high-quality images for production.

### Logos
- `logo-full.svg` - Full wordmark logo with icon
- `logo-icon.svg` - Icon-only version for compact spaces

### Favicons
- `favicon-16x16.png` - 16×16px favicon
- `favicon-32x32.png` - 32×32px favicon
- `apple-touch-icon.png` - 180×180px iOS home screen icon

### Feature Screenshots (Placeholders)
- `hero-dashboard-mockup.png` - Main hero section dashboard (1200×900px)
- `feature-client-management.png` - Client management feature (800×600px)
- `feature-roadmaps.png` - Roadmap feature showcase (800×600px)
- `feature-analytics.png` - Analytics dashboard (800×600px)

### Social Media
- `og-image.jpg` - Open Graph image for social sharing (1200×630px)

## Replacing Placeholders

To replace these placeholder images with actual assets:

1. **Take real screenshots** from your Sage.coach application
2. **Export at the correct dimensions** (see filenames above)
3. **Optimize images** using tools like TinyPNG or Squoosh
4. **Replace files** with the same filename
5. **Test on the live site** to ensure they display correctly

## Asset Requirements

For detailed specifications on image dimensions, formats, and optimization guidelines, see:
`/docs/ASSET_REQUIREMENTS.md`

## Notes

- All placeholder images use the Sage brand colors (#355C4C, #A3B9A0)
- SVG files are used where possible for scalability
- Images include proper alt text in the HTML for accessibility
- Lazy loading is enabled for below-the-fold images
